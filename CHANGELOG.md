## [未リリース] — 2026-07-09 (v20260709n)

### 追加
- **詳細タブ未入力バッジ** — 未入力の必須項目があるタブに赤バッジ `(N)` を表示（どのタブを開けばいいか即判断）

- **商品詳細完成度バナー** — 詳細ページのタブ上部に完成度バーと「未入力: 製造者名・製造者住所」を表示（何を入力すべきか一目で分かる）
- **商品並び替えオプション** — 商品一覧ツールバーに「更新日・名前順・完成度順」切替セレクトを追加 (`masterSort` 状態 + `data-master-sort`)
- **賞味期限アラートバナー** — ダッシュボードに期限切れ（赤）/ 30日以内（黄）のアラートバナーを追加
- **賞味期限日フィールド（管理用）** — 商品詳細 基本タブの「賞味期限」直下に ISO 日付の管理用フィールドを追加（ラベル非出力）
- **期限切れフィルター** — 商品一覧に `expired` / `expiringSoon` フィルター対応・TODO_LABELS に追加
- **検索ボックス ✕ クリアボタン** — 検索中にワンクリックでクリア・フォーカス復帰 (`data-clear-search`)
- **キーボードショートカットヒントバー** — 商品一覧ツールバー下に `/` · `N` · `Ctrl+S` ヒントを表示（デスクトップのみ）
- **`showModal` dangerLabel 第3ボタン** — 破壊的操作を赤ボタンで明示する3択モーダルに対応
- **`calcTodo(derivedAll?)` 引数追加** — 呼び出し元から derive 済みデータを渡して重複計算を回避
- **ダッシュボード「最近の商品」完成度 % 表示** — derivedAll を流用してコスト 0 で追加
- **プロジェクト管理ファイル群** — ROADMAP.md / TODO.md / CHANGELOG.md / PROJECT_RULES.md / README.md 新規作成

### 修正
- **JSONインポートモーダル背景クリックバグ** — `showModal` の背景クリックが `onCancel()` を発火し全データ置き換えが実行されていた → `close()` のみに修正
- **「登録商品数」stat-card クリック後フィルター残存** — `data-set-filter="all"` を追加して全件表示に戻るよう修正
- **JSONインポート UX** — `onCancel` に replace を割り当てていた設計ミスを `dangerLabel/onDanger` で修正し「キャンセル」を追加

### 改善
- **商品削除フロー** — 確認モーダル → 即削除 + 5秒 undo トーストに変更（誤削除リカバリー可能）
- **タブ切替スクロールリセット** — `data-detail-tab` クリック後に `.saas-content` を `scrollTo(0,0)`
- **フィルター結果 0 件メッセージ** — TODO フィルター適用時は「✅ 該当なし」、検索 0 件は「一致なし」に分岐
