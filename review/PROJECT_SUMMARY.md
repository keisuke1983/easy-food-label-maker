# FoodPilot — プロジェクト概要

> レビュー用サマリー（2026-07-11 時点）

---

## 1. FoodPilotとは何か

**FoodPilot（フードパイロット）** は、食品メーカー・小規模食品事業者向けに特化した AI 搭載の商品管理 SaaS です。

食品表示法に準拠したラベル作成・原材料管理・規格書生成・AI相談・在庫管理まで、**食品業務のバックオフィスをワンストップで完結**できるウェブアプリケーションです。

---

## 2. コンセプト

> **「FoodPilot = 食品メーカーのAI商品管理責任者」**

専門知識がなくても食品表示法に準拠した商品管理ができる。AIがアドバイザーとして常駐し、ラベル確認・説明文生成・表示チェック・在庫更新まで自動化する。

---

## 3. ターゲット

| セグメント | 規模感 |
|-----------|--------|
| 食品メーカー（菓子・惣菜・パン等） | 年商 〜3億円 |
| 農家直売所・道の駅 | 個人〜数名 |
| 菓子製造業者・惣菜店 | 小規模事業者 |
| 食品 EC 販売者 | 楽天・Amazon 出店者 |

共通の課題: 食品表示法の専門知識不足・ラベル作成の手間・規格書整備コスト・在庫管理のアナログ運用

## 5. 特徴・強み

### 食品業界特化
- 食品表示法準拠のリアルタイムラベルチェック
- 28品目アレルゲン自動検出
- 原材料配合順（重量降順）自動チェック
- 添加物キーワード入り名称の警告
- 原産地表示警告（農産物・畜産物・水産物キーワード検出）

### AIネイティブ設計
- AIからのダッシュボード能動提案（賞味期限切れアラート・原価未設定警告等）
- AI説明文生成（楽天・Amazon・LINE等チャネル別）
- AI棚スキャン β（写真から在庫を自動認識）
- AI相談チャット（商品ごとのコンテキスト付き）

### 優れたUX
- ゼロインストール（ブラウザのみ動作）
- PWA対応（ホーム画面追加・オフライン閲覧）
- 自動保存（2秒デバウンス）
- 商品削除 5秒 undo トースト
- キーボードショートカット（/ N Ctrl+S Esc）
- 保存済み検索プリセット
- 複合フィルター（カテゴリ AND 完成度 AND ステータス AND 課題）

### データ安全性
- localStorage 保存（データがサーバーに送られない）
- JSON フルバックアップ（画像含む/除く選択）・CSV エクスポート/インポート
- 変更履歴スナップショット・復元機能（最大10件）

## 6. 競合との差別化

| 競合 | 弱点 | FoodPilot の優位性 |
|------|------|-------------------|
| Excel / Googleスプレッドシート | ラベル出力不可・アレルゲン手動管理 | 自動チェック・PDF出力・AI相談 |
| 汎用ラベル作成ソフト | 食品表示法非対応・高価 | 食品特化・低価格・ブラウザ動作 |
| 大手ERPシステム | 高額・導入ハードルが高い | 無料プランあり・即時利用開始 |
| ラベル屋さん等 | 管理機能なし・印刷のみ | 商品マスター・規格書・AI一体化 |

## 7. 技術構成

```
フロントエンド: バニラ JS SPA（フレームワーク不使用）
  - グローバル状態 + render() によるフル再描画
  - イベントデリゲーション（setupDelegation / bindDynamic）
  - PWA（manifest.json + Service Worker）
  - スクリプトロード順: db.js → constants.js → utils-core.js → state.js → delegation.js → app.js

バックエンド: Vercel Serverless Functions（api/ai-photo.js）
  - 環境変数: GEMINI_API_KEY（準備中）

データ永続化: localStorage（メイン）/ sessionStorage（セッション限定）

インフラ: Vercel（静的ホスティング + Serverless）
  - URL: https://easy-food-label-maker.vercel.app
  - GitHub: https://github.com/keisuke1983/easy-food-label-maker (branch: main-update)

外部サービス（未接続 or 準備中）:
  - Google Gemini 2.0 Flash（棚スキャン・写真登録 準備中）
  - Supabase（クラウド同期 UI実装済み・未接続）
```
