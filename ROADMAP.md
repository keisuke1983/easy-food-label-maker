# ROADMAP.md — 食品商品管理クラウド 開発ロードマップ

---

## プロジェクト概要

| 項目 | 内容 |
|------|------|
| **アプリ名** | 食品商品管理クラウド |
| **コンセプト** | 食品メーカー・小規模食品事業者向け AI 搭載商品管理 SaaS |
| **ターゲット** | 年商〜3億円規模の食品メーカー、農家直売所、菓子製造業者、惣菜店 |
| **提供価値** | 食品表示法に準拠したラベル作成・原材料管理・規格書作成を、専門知識なしで完結できる |
| **最終目標** | 日本で最も使いやすい食品業界向け AI SaaS として月額 3,980 円以上でも継続契約されるプロダクト |
| **GitHub** | https://github.com/keisuke1983/easy-food-label-maker |
